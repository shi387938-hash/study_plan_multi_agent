# 多Agent协同学习/备考计划定制系统

这是一个基于多智能体（Multi-Agent）协作的自动化学习规划系统，专为学生备考场景设计。系统通过 4 个专业 Agent 的协同工作，实现从学情分析、计划生成到进度跟踪、动态调整的全流程自动化。

## ✨ 系统特性

- 🤖 **多Agent协同**：4个专业智能体分工协作，模拟专家团队为你服务
- 📊 **智能学情分析**：自动分析你的基础水平、薄弱环节
- 📋 **个性化计划生成**：根据你的情况生成科学的分阶段学习计划
- 🔄 **动态计划调整**：根据每日学习进度自动优化后续计划
- 💾 **本地数据存储**：所有学习数据保存在本地，保护隐私
- 🔌 **灵活模型支持**：支持 OpenAI、Ollama 本地模型、豆包 / 通义千问等兼容接口

## 🏗️ 系统架构

本系统包含 4 个专业 Agent，它们各司其职，协同完成学习规划任务：

| Agent | 核心职责 |
|-------|----------|
| **学情分析专家** | 分析你的成绩、时间、目标，生成精准的学情画像 |
| **学习计划规划师** | 根据学情画像，生成分阶段、可执行的个性化学习计划 |
| **进度跟踪分析师** | 跟踪每日打卡数据，复盘学习效果，识别薄弱环节 |
| **计划动态调整师** | 根据进度反馈，自动优化调整后续学习计划 |
