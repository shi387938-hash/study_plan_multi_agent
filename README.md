🎯 多 Agent 协同学习 / 备考计划定制系统
这是一个基于多智能体（Multi-Agent）协作的自动化学习规划系统，专为学生备考场景设计。系统通过 4 个专业 Agent 的协同工作，实现从学情分析、计划生成到进度跟踪、动态调整的全流程自动化。
✨ 系统特性
🤖 多 Agent 协同：4 个专业智能体分工协作，模拟专家团队为你服务
📊 智能学情分析：自动分析你的基础水平、薄弱环节
📋 个性化计划生成：根据你的情况生成科学的分阶段学习计划
🔄 动态计划调整：根据每日学习进度自动优化后续计划
💾 本地数据存储：所有学习数据保存在本地，保护隐私
🔌 灵活模型支持：支持 OpenAI、Ollama 本地模型、豆包 / 通义千问等兼容接口
🏗️ 系统架构
本系统包含 4 个专业 Agent，它们各司其职，协同完成学习规划任务：
Agent
职责
学情分析专家
分析你的成绩、时间、目标，生成精准的学情画像
学习计划规划师
根据学情画像，生成分阶段、可执行的个性化学习计划
进度跟踪分析师
跟踪每日打卡数据，复盘学习效果，识别薄弱环节
计划动态调整师
根据进度反馈，自动优化调整后续学习计划

🚀 快速开始
1. 环境准备
首先安装依赖包：
pip install -r requirements.txt

2. 配置大模型 API
系统支持多种大模型，你可以选择其中一种：
方式一：使用 OpenAI API
export OPENAI_API_KEY="你的OpenAI密钥"

方式二：使用本地 Ollama 模型（推荐，完全免费本地运行）
先安装并启动 Ollama：https://ollama.com/
拉取你需要的模型，比如：ollama pull qwen2:7b
然后在代码的 LLM_CONFIG 中修改：
LLM_CONFIG = {
    "config_list": [
        {
            "model": "qwen2:7b",  # 你拉取的模型名
            "api_key": "ollama",  # 本地模型随便填
            "base_url": "http://localhost:11434/v1",
        }
    ],
    ...
}

方式三：使用国内大模型（豆包 / 通义千问等）
这些模型都兼容 OpenAI 接口格式，只需修改 base_url 和 api_key 即可：
# 例如豆包API
LLM_CONFIG = {
    "config_list": [
        {
            "model": "doubao-pro",
            "api_key": "你的豆包API密钥",
            "base_url": "https://ark.cn-beijing.volces.com/api/v3",
        }
    ],
    ...
}

3. 运行系统
python study_plan_multi_agent.py

📖 使用指南
第一步：创建学习计划
首次运行请先创建你的个性化学习计划：
选择功能 1
输入考试信息（考试名称、剩余天数、每日可用时间）
输入各科目的当前成绩和目标成绩
系统会自动启动多 Agent 协作，为你生成专属学习计划！
示例输入：
考试名称：考研英语
距离考试还有多少天？: 90
你每天可用的学习时间是多少小时？: 6
科目名称: 词汇
当前成绩: 40
目标成绩: 70
科目名称: 阅读
当前成绩: 50
目标成绩: 80
科目名称: q

第二步：每日学习打卡
每天学习完成后，进行打卡：
选择功能 2
输入今日学习的科目、时长、内容、测试成绩
系统会自动分析你的进度，并动态调整后续的学习计划！
第三步：查看学习进度
随时可以查看你的学习总览：
选择功能 3
查看累计学习时长、平均每日学习时间、打卡记录
💾 数据存储
所有你的学习数据都会保存在本地的 study_plan_data.json 文件中，包括：
你的学习计划
每日打卡记录
各科目的进度
数据完全保存在你的本地电脑，不会上传到任何服务器，保护你的隐私。
